My Blog

This blog documents my progress and thoughts as I learn more about design and web development. Here are some entries from my journey.

Reflections

Thoughts and experiences from my learning journey.

Learning HTML

I started by learning basic HTML tags like headings, paragraphs, links, and lists. It was exciting to build structure for a real website.

Getting Started with CSS

CSS helped me make my site look much better. I learned about colors, spacing, and fonts. It’s like painting over your HTML framework.

Making My First Website

My first website was basic but felt like a huge milestone. I used a combination of HTML and CSS to make a simple but functional page.

Graphic Design Practice

Using Photoshop, I tried editing images and playing with color and text effects. It gave me a better eye for layout and balance.

Learning Dreamweaver

I used Dreamweaver to code my pages and preview them live. It helped a lot with keeping my HTML and CSS organized and clean. It also saved a lot of time as I didn't need to keep going back and refreshing the page on the browser

How I Made My Website Mobile-Friendly

I used media queries and flexible widths so the site would look good on both phones and laptops. It took some trial and error but was worth it.

Comparing Early vs. Recent Projects

Looking at my old sites like Robocat and comparing them to Sweet Treats Bakery, I can see how much I’ve improved. It’s encouraging to see progress.

What I’d Do Differently Next Time

Next time, I’d spend more time planning layouts and use more consistent styles. I’ve learned that thinking ahead saves a lot of fixing later.

Picking Fonts and Colors That Work Together Nicely

When I first started, the colors didn't match well together. It looked sloppy and honestly needed a lot of work. I also didn't think much about fonts as I thought having the default font was okay. It wasn't later until I realized how important fonts were, especially if I wanted to page to look nice.
